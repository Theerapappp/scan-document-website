# Document Image to Text

## 🛠 เทคโนโลยีที่ใช้

- **Frontend Framework**: [React 19]
- **Build Tool**: [Vite]
- **Styling**: [Tailwind CSS]
- **Icons**: [Lucide React]
- **AI Model**: Google Gemini API (`gemini-2.5-flash`)

## ✨ ฟีเจอร์หลัก (Features)

- **อัปโหลดรูปภาพ**: รองรับไฟล์ภาพเอกสารนามสกุล PNG, JPG และ JPEG
- **ครอบตัดรูปภาพ (Crop)**: สามารถวาดกรอบเพื่อเลือกวิเคราะห์เฉพาะส่วนของข้อความที่ต้องการบนเอกสารได้
- **วิเคราะห์เอกสารด้วย AI**: ดึงข้อความและจัดหมวดหมู่โครงสร้างเอกสารโดยอัตโนมัติ (รองรับหลายภาษา รวมทั้งภาษาไทย)
- **แก้ไขและจัดการผลลัพธ์**: 
  - แก้ไขประเภทของเอกสาร (Document Type)
  - แก้ไขชื่อหมวดหมู่และเนื้อหาที่ AI ดึงข้อมูลมา
  - สามารถลบหรือเพิ่มหมวดหมู่ใหม่ได้ตามต้องการ
- **คัดลอกข้อมูลรวดเร็ว**: มีปุ่ม "Copy All" สำหรับคัดลอกผลลัพธ์ทั้งหมดเพื่อนำไปวางในโปรแกรมอื่น เช่น Word หรือ Notepad ได้ทันที

## 🚀 วิธีการติดตั้งและรันโปรเจค

### 1. โคลนหรือดาวน์โหลดโปรเจค
เข้าไปที่โฟลเดอร์โปรเจค `scandocument` ผ่าน Command Line:
```bash
cd scandocument
```

### 2. ติดตั้ง Dependencies
รันคำสั่งด้านล่างเพื่อติดตั้งแพ็กเกจที่จำเป็นทั้งหมด (เช่น React, Lucide):
```bash
npm install
```

### 3. ตั้งค่า API Key ของ Gemini
โปรเจคนี้จำเป็นต้องใช้ Gemini API ในการทำงาน 
1. ไปที่เว็บไซต์ [Google AI Studio](https://aistudio.google.com/app/apikey) และสร้าง API Key ของตัวเอง
2. เปิดไฟล์ `src/App.jsx`
3. ค้นหาโค้ดในบรรทัดที่ 7 นำ API Key ของคุณมาใส่แทนที่ `"YOUR_API_KEY"`:
   ```javascript
   const apiKey = "YOUR_API_KEY"; // เอา API Key ที่ได้มาวางตรงนี้
   ```

### 4. รันโปรเจค
ทดสอบรันแอปพลิเคชันในโหมด Development:
```bash
npm run dev
```
เมื่อรันสำเร็จ ให้เปิดเว็บเบราว์เซอร์ไปที่ URL ที่แสดงใน Terminal (โดยปกติจะเป็น `http://localhost:5173`)

## 💡 วิธีการใช้งาน

1. **อัปโหลดรูปภาพ**: ในหน้าเว็บฝั่งซ้าย คลิกที่กล่อง **"Upload & Crop"** เพื่อเลือกไฟล์ภาพเอกสารที่ต้องการ
2. **เลือกพื้นที่ (ถ้าต้องการ)**: ลากเมาส์ครอบพื้นที่บนรูปภาพที่ต้องการให้ AI วิเคราะห์ (หากต้องการให้วิเคราะห์ทั้งภาพ ก็ไม่ต้องวาดกรอบใดๆ)
3. **ประมวลผล**: กดปุ่มสีดำ **"Extract & Categorize Text"** แล้วรอระบบประมวลผลสักครู่
4. **ตรวจสอบผลลัพธ์**: ข้อมูลที่ดึงได้จะแสดงทางฝั่งขวา โดยแยกตามหมวดหมู่โครงสร้างของเอกสาร
5. **ปรับแต่ง**: สามารถแก้ไขข้อความ เปลี่ยนชื่อหมวดหมู่ ลบ/เพิ่มหมวดหมู่ใหม่ ได้ง่ายๆ เพียงแค่คลิกและพิมพ์
6. **คัดลอกไปใช้งาน**: เมื่อพอใจกับข้อความแล้ว สามารถกดปุ่ม **"Copy All"** มุมขวาบนของส่วนผลลัพธ์ เพื่อนำข้อความทั้งหมดไปใช้งานต่อได้ทันที
